Volunteer Page

Technical Description

On this webpage, I divided the content into several sections using div elements. In the first section, I featured a looping video with text and links overlaid on top. I accomplished this by applying CSS properties such as position: relative and overflow: hidden, while the video plays automatically, loops, and remains muted. A semi-transparent black overlay improves text visibility by using position: absolute, full width and height, and a background color of rgba(0, 0, 0, 0.2). Additionally, I overlaid a darker plane at the bottom that includes text and a button which scrolls to the relevant section. The button changes color on hover using the :hover selector and has rounded edges with border-radius.

The next section focuses on why people should volunteer. It features a background image set in a div with width: auto, height: 100%, and position: absolute. Over this image, I added a semi-transparent white overlay by using another div that takes up 65% of the width and adjusts automatically to different screen sizes, with the background color set to rgba(255, 255, 255, 0.6). The overlay's text is centered using text-align: center and position: relative, styled with a font size of 23px and margins on both sides for spacing.

In the following section, I added a sort-by button using select and option tags. This button is positioned in the top-right corner with top: 20px and right: 20px, styled with padding: 6px 12px and a cursor: pointer to indicate interactivity. On the left side, there is text showing the number of available volunteer programs along with a "View All" link. Below these elements, I created four program cards using div elements with the same class. Each card features images with a hover effect that slightly enlarges the card using the CSS rule .program-card:hover { transform: scale(1.05); }. Inside each card, there are two buttons labeled "View Project" and "Apply," as well as a like button that toggles to red when clicked, achieved using a JavaScript function that toggles a "liked" class.

For the final section, I created a review area divided into two parts using div elements. The left side occupies 30% of the width while the right side takes up 70%, ensuring a consistent layout across devices. The right side contains two sample review boxes, and at the bottom, a text box is provided for users to type and submit their reviews. This text box is implemented using a textarea element with a placeholder that reads "Write your review here‚Ä¶".
